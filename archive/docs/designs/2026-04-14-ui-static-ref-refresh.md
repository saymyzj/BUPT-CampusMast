# `frontend/ui-static/ref` 视觉重设计说明

## 背景

当前 `frontend/ui-static/ref` 下的 5 套静态参考页已经具备较完整的信息结构，但其中部分页面仍然明显依赖高饱和度渐变、玻璃拟态或通用化的 AI 风格装饰，辨识度不够稳定。用户希望在不改动整体结构和内容的前提下，把这些页面优化成更有记忆点、并且更适合后续真实产品落地的参考稿。

## 目标

- 保留 5 套页面现有的信息层次和主要结构
- 强化每套方案的独立视觉语言，让它们看起来像 5 个明确的设计方向
- 避免“高饱和渐变 + 模糊玻璃”的常见 AI 模板感
- 让 A 同学后续拿这些页面做 Vue 落地时，更容易提炼组件气质

## 非目标

- 不改页面文案和业务结构
- 不新增复杂交互逻辑
- 不把静态页重构成组件化前端代码

## 方案

本次优化把 5 套参考页分别收敛到以下方向：

1. `style-1-fresh-academic.html`
   - 方向：校园工坊 / 纸张与印章感
   - 关键词：米白纸面、钴蓝重点色、低饱和红章、卡片像任务便签

2. `style-2-dark-geek.html`
   - 方向：实验室终端 / 夜间控制台
   - 关键词：炭黑、磷光绿、信号黄、细网格、冷静而克制

3. `style-3-business-elite.html`
   - 方向：校内运营驾驶舱 / 黑金报表
   - 关键词：石灰白、墨黑、铜金、强秩序感、地图像印刷版式

4. `style-4-vibrant-campus.html`
   - 方向：校园海报 / Riso 印刷感
   - 关键词：奶油底、群青、番茄红、亮黄、图形块面，而非渐变

5. `style-5-nordic-minimal.html`
   - 方向：展览标签 / 编辑部极简版式
   - 关键词：骨白、鼠尾草、炭灰、留白、细线和排版张力

## 验证

- 页面仍可直接作为独立 HTML 打开
- 页面结构和主要内容未改动
- 每个页面至少有清晰且独立的配色、背景、卡片和导航气质
- 高饱和渐变被明显削弱或移除
